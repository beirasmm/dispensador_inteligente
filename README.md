# Dispensador de agua inteligente _SmartRefill_

## Descripción del proyecto
El proyecto _SmartRefill_ consiste en el desarrollo de un dispensador de agua inteligente destinado para el uso doméstico o de oficina. El dispositivo detecta automáticamente la presencia de un recipiente y lo rellena con la cantidad necesaria. El objetivo principal del dispositivo es ofrecer una solución cómoda y automatizada para el día a día, garantizando la eficiencia y responsabilidad en el consumo de auga.
